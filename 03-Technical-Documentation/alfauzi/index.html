Apa itu Bahasa Pemrograman Python?

Python merupakan bahasa pemrograman tingkat tinggi dan merupakan salah satu bahasa pemrograman terpopuler di dunia. Python adalah general purpose programming language yang dapat digunakan untuk berbagai aplikasi seperti Artificial Intelligence, IoT, Data Science, Web Development, hingga Automation.

Python sangat diminati karena fleksibilitasnya dan bisa digunakan di berbagai platform dan kasus penggunaan.

Contoh Aplikasi Python: Web Development dengan Django atau Flask, Analisis Data dengan Pandas, dan Machine Learning dengan TensorFlow.

Kenapa Python Begitu Terkenal?

Python terkenal karena sintaksnya yang sederhana dan menyerupai bahasa manusia, membuatnya mudah dipelajari oleh pemula. Bahasa ini juga memiliki berbagai library dan tools yang mendukung pengembangan aplikasi di banyak bidang.

Contoh sintaks Python:

if 4 in [1,2,3,4]:
    print("Terdapat angka 4!")

Python juga menyediakan library seperti pandas untuk manipulasi data, TensorFlow untuk machine learning, dan banyak lagi, membuatnya ideal untuk berbagai bidang.

Tipe Data dalam Python

Python memiliki beberapa tipe data dasar yang berperilaku berbeda. Tipe data ini penting karena menentukan bagaimana operasi dilakukan terhadap nilai yang disimpan dalam variabel.

Jenis Tipe Data:

  • Number: Angka, bisa berupa bilangan bulat (int) atau desimal (float).
  • Boolean: Nilai logika, yaitu True atau False.
  • String: Sekumpulan karakter yang diapit tanda kutip, digunakan untuk teks.
  • List: Sekumpulan item yang dapat diubah, disusun dalam urutan, dan dikelilingi oleh tanda kurung siku.
  • Tuple: Sama seperti list, tetapi tidak dapat diubah (immutable) dan menggunakan tanda kurung biasa.
  • Dictionary: Struktur data yang menyimpan pasangan key-value yang dapat diakses berdasarkan kunci.

Contoh:

python
# Mengecek tipe data
x = 10
print(type(x))  # Output: 

# Mengubah tipe data
y = "10"
print(int(y))  # Output: 10
Tipe Data String

String di Python adalah kumpulan karakter yang dapat dimanipulasi menggunakan berbagai operasi seperti penjumlahan, perkalian, dan pengindeksan. String juga mendukung banyak fungsi bawaan.

Contoh:

python
# Pengindeksan string
s = "Python"
print(s[0])  # Output: 'P'
print(s[-1])  # Output: 'n'

# Fungsi bawaan
teks = "Halo semua"
print(teks.split())  # Output: ['Halo', 'semua']
Operator dalam Python

Operator adalah simbol yang digunakan untuk melakukan operasi pada variabel dan nilai. Python memiliki beberapa jenis operator yang dapat digunakan dalam berbagai konteks.

Jenis-Jenis Operator:

  • Operator Aritmatika: Digunakan untuk operasi matematika, seperti penjumlahan (+), pengurangan (-), perkalian (*), dan pembagian (/).
  • Operator Perbandingan: Digunakan untuk membandingkan dua nilai, seperti sama dengan (==), lebih besar (>) dan lebih kecil (<).
  • Operator Logika: Digunakan untuk operasi logika seperti and, or, dan not.
  • Operator Penugasan: Operator seperti = digunakan untuk menetapkan nilai ke variabel.

Contoh:

python
x = 10
y = 5
print(x + y)  # Output: 15
print(x > y)  # Output: True
Variabel dan Identifier

Variabel digunakan untuk menyimpan nilai, dan identifier adalah nama yang diberikan untuk entitas seperti variabel dan fungsi. Identifier harus mematuhi aturan tertentu, seperti tidak boleh diawali dengan angka dan tidak boleh menggunakan kata kunci Python.

Contoh:

python
lebar = 10
tinggi = 5
luas = lebar * tinggi
print("Luas =", luas)  # Output: Luas = 50
Percabangan If Else

Percabangan digunakan untuk mengeksekusi kode berdasarkan kondisi tertentu. Struktur if-else memungkinkan program memilih blok kode yang akan dijalankan berdasarkan kondisi logika.

Contoh:

python
umur = 18
if umur < 20:
    print("Youth discount")  # Output: Youth discount
elif umur >= 20:
    print("No discount")
Looping For While

Python mendukung dua jenis loop utama: for dan while, yang memungkinkan pengulangan eksekusi blok kode.

Contoh For Loop:

python
# For loop
for i in range(5):
    print("Hello")  # Output: Hello (diulang 5 kali)

Contoh While Loop:

python
# While loop
i = 0
while i < 5:
    print(i)  # Output: 0, 1, 2, 3, 4
    i += 1
Break dan Continue

break digunakan untuk menghentikan loop secara langsung, sedangkan continue digunakan untuk melewati satu iterasi dan melanjutkan ke iterasi berikutnya.

Contoh Break:

python
for i in range(10):
    if i == 5:
        break  # Menghentikan loop saat i = 5
    print(i)

Contoh Continue:

python
for i in range(10):
    if i == 5:
        continue  # Melewati iterasi saat i = 5
    print(i)
Mengambil Input dari Pengguna

Fungsi input() digunakan untuk mengambil input dari pengguna. Input yang diambil dari input() selalu dalam bentuk string, jadi jika dibutuhkan input numerik, Anda harus melakukan konversi tipe data menggunakan int() atau float().

Contoh:

python
x = int(input("Masukkan angka: "))
print("Angka yang dimasukkan adalah", x)
